Sticker Maker v3.3 - Справка

ОСНОВНЫЕ ВОЗМОЖНОСТИ:
• Создание стикеров из Excel накладных
• Автоматический поиск артикулов в базе
• Поддержка Code128 и QR-кодов
• Настраиваемый дизайн стикеров
• Live-предпросмотр изменений
• Управление базой адресов хранения

ПОРЯДОК РАБОТЫ:
1. Во вкладке "База данных" загрузите номенклатуру
2. Настройте внешний вид стикера во вкладке "Настройки"
3. Выберите файл накладной во вкладке "Обработка"
4. Нажмите "Создать стикеры"

НАСТРОЙКИ СТИКЕРА:
• Размер и ориентация
• Шрифты артикула, названия, количества, адреса
• Цвета и выравнивание
• Позиция и тип кода (QR/Code128)
• Автоматические правила для кодов

CODE128 ПОДДЕРЖКА:
• Поддерживает все печатные ASCII символы (32-126)
• Автоматически переключается на QR при недопустимых символах
• Можно отключить в настройках

БАЗА ДАННЫХ:
• Загружается из CSV файла по URL
• Автоматически обновляется при запуске
• Поиск артикулов по точному совпадению и нечеткому поиску

БАЗА АДРЕСОВ:
• Импорт из Excel файлов
• Поиск адресов хранения по артикулу
• Автоматическое отображение адресов на стикерах

ФАЙЛ КОНФИГУРАЦИИ:
{appdata}\StickerMakerV3\config.json

ГОРЯЧИЕ КЛАВИШИ:
• Ctrl+S - Сохранить настройки
• Ctrl+C - Копировать текст
• Ctrl+V - Вставить текст

ПРОБЛЕМЫ:
• Если не создаются стикеры - проверьте базу данных
• Если не читается файл - проверьте формат Excel
• Если код не создается - проверьте символы в артикуле
• Если не виден предпросмотр - проверьте настройки шрифтов
