# ScanHead Combine

Модульное ядро для системы инвентаризации и управления складом. Версия 2.0.

## 🎯 Цель

Разделение монолитной логики legacy-проектов на независимые модули с четкими интерфейсами. Сборка нового приложения `scanhead-combine` на основе этих модулей.

## 📦 Структура

```
├── libs/                   # Библиотеки (модули)
│   ├── core/               # Инфраструктура: bootstrap, инициализация
│   ├── i18n/               # Интернационализация
│   ├── domain/             # Доменная логика
│   ├── repository/         # Доступ к данным
│   ├── scanner_input/      # Обработка ввода сканеров
│   ├── printing/           # Генерация этикеток
│   ├── config/             # Управление настройками
│   ├── utils/              # Утилиты
│   └── ...                 # Другие модули
│
├── gui/                    # GUI приложение (customtkinter)
│   ├── main_window.py      # Корневое окно
│   ├── search_bar.py       # Поиск с debounce и подсказками
│   ├── product_details.py  # Детали товара
│   ├── tabs/               # Вкладки
│   ├── dialogs/            # Диалоги
│   └── widgets/            # Кастомные виджеты
│
├── services/               # Сервисный слой
├── config/                 # Конфигурация
└── main.py                 # Точка входа
```
