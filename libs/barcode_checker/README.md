# Barcode Checker Module

Версия: **0.1.0**

## Назначение
Модуль для проверки, валидации и обработки штрих-кодов различных форматов.

## Установка
```bash
# Модуль является частью проекта и не требует отдельной установки
from libs.barcode_checker import BarcodeChecker
```

## Использование

### Валидация EAN-13
```python
checker = BarcodeChecker()
is_valid = checker.validate_ean13("4601234567890")
# Результат: True/False
```

### Вычисление контрольной цифры
```python
check_digit = checker.calculate_ean13_check_digit("460123456789")
# Результат: 0 (контрольная цифра для этого префикса)
```

### Определение формата
```python
format_type = checker.detect_format("4601234567890")
# Результат: "EAN-13"
```

### Нормализация
```python
normalized = checker.normalize("4601-2345-6789-0")
# Результат: "4601234567890"
```

## API

### Класс `BarcodeChecker`

#### Методы
- `validate_ean13(barcode: str) -> bool` - Проверка EAN-13 по контрольной сумме
- `calculate_ean13_check_digit(barcode_12: str) -> Optional[int]` - Вычисление контрольной цифры
- `validate_ean8(barcode: str) -> bool` - Проверка EAN-8 (TODO)
- `validate_upc_a(barcode: str) -> bool` - Проверка UPC-A (TODO)
- `validate_code128(barcode: str) -> bool` - Проверка Code-128 (TODO)
- `normalize(barcode: str) -> str` - Нормализация штрих-кода
- `detect_format(barcode: str) -> Optional[str]` - Определение формата
- `get_supported_formats() -> List[str]` - Список поддерживаемых форматов

## Поддерживаемые форматы

| Формат | Длина | Статус |
|--------|-------|--------|
| EAN-13 | 13 цифр | ✅ Реализовано |
| EAN-8 | 8 цифр | ⏳ В разработке |
| UPC-A | 12 цифр | ⏳ В разработке |
| Code-128 | Переменная | ⏳ В разработке |

## Известные проблемы и план доработки

### Текущие ограничения
1. **Частичная реализация**: Работает только валидация EAN-13
2. **Отсутствует проверка EAN-8**: Метод выбрасывает `NotImplementedError`
3. **Отсутствует проверка UPC-A**: Метод выбрасывает `NotImplementedError`
4. **Отсутствует проверка Code-128**: Метод выбрасывает `NotImplementedError`

### План доработки (v0.2.0)
- [ ] Реализовать алгоритм проверки EAN-8
- [ ] Реализовать алгоритм проверки UPC-A (аналогичен EAN-13)
- [ ] Реализовать проверку Code-128 (требует работы с символьными наборами)
- [ ] Добавить поддержку генерации штрих-кодов
- [ ] Добавить проверку префиксов стран производителей

### Баги, требующие исправления
- Нет обработки нецифровых символов в режиме строгой валидации
- Отсутствует поддержка штрих-кодов с дополнительными символами (добавки 2/5 цифр)

## Алгоритм проверки EAN-13

Контрольная сумма вычисляется следующим образом:
1. Сложить цифры на нечетных позициях (1, 3, 5, 7, 9, 11)
2. Сложить цифры на четных позициях (2, 4, 6, 8, 10, 12) и умножить на 3
3. Сложить результаты шагов 1 и 2
4. Контрольная цифра = (10 - (сумма % 10)) % 10

## Тестирование
```python
def test_ean13():
    checker = BarcodeChecker()
    assert checker.validate_ean13("4601234567890") is True
    assert checker.validate_ean13("4601234567891") is False
    assert checker.calculate_ean13_check_digit("460123456789") == 0
    assert checker.detect_format("4601234567890") == "EAN-13"
```
