# Parsers Module (libs/parsers)

Парсеры внешних форматов данных.

## Состав

Модуль находится в разработке. Планируется:

### invoice_parser.py
Парсер накладных и товарно-сопроводительных документов.
- Поддержка форматов: Excel, CSV, XML (1С)
- Извлечение позиций, количеств, цен
- Нормализация артикулов

### waybill_parser.py
Парсер путевых листов и транспортных накладных.

## Зависимости
- `pandas` — для работы с табличными данными
- `openpyxl` — для чтения Excel

## Примечания
- Все парсеры возвращают стандартизированные dataclass-модели
- Обработка ошибок валидации входных данных
